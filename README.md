# 🐔 Country Chicken Farm Manager

A simple, lightweight web app to manage your country chicken farm — daily todos, income tracking, and expense tracking.

**No server needed. No login. No cost. Your data stays in your browser.**

---

## 🚀 How to Deploy on GitHub Pages (Free Hosting)

### Step 1: Create a GitHub Repository

1. Go to [github.com](https://github.com) and log in
2. Click the **+** button (top right) → **New repository**
3. Name it: `chicken-farm-manager`
4. Set it to **Public**
5. Click **Create repository**

### Step 2: Upload Files

1. On your new repository page, click **"uploading an existing file"** link
2. Drag and drop these 3 files:
   - `index.html`
   - `style.css`
   - `app.js`
3. Click **Commit changes**

### Step 3: Enable GitHub Pages

1. Go to your repository → **Settings** tab
2. In the left sidebar, click **Pages**
3. Under **Source**, select **Deploy from a branch**
4. Under **Branch**, select `main` and folder `/` (root)
5. Click **Save**
6. Wait 1-2 minutes, then refresh the page

### Step 4: Access Your App

Your app will be live at:
```
https://YOUR-USERNAME.github.io/chicken-farm-manager/
```

Bookmark this URL on your phone for easy access!

---

## 💾 Data Backup

Your data is stored in your browser. To keep it safe:

1. Open the **Backup** tab in the app
2. Click **Export Data** to download a backup file
3. Store the backup file somewhere safe (Google Drive, email it to yourself, etc.)
4. If you ever lose data, use **Import Data** to restore it

> ⚠️ **Important**: If you clear your browser data or switch devices, you'll need to import your backup file. Export your data regularly!
